# 집계

#### NULL

```sql
SELECT COALESCE(COLUMN, NULL을 대체할 값)
```

# 비교/범위

#### 날짜

```sql
WHERE b.CREATED_DATE BETWEEN '2022-10-01' AND '2022-10-31'
# WHERE SUBSTR(b.CREATED_DATE, 1, 7) = '2022-10'
```
